### S3 Bucket

To connect to an Amazon S3 bucket:

<Steps>
  <Step>Go to the 'Links' page on your Structured dashboard. </Step>
  <Step>
    Choose 'S3 Bucket' and enter your AWS Access Key ID and Secret Access Key.
  </Step>
  <Step>
    Provide the name of your S3 bucket and the corresponding AWS region.
  </Step>
  <Step>
    Confirm the permissions to ensure Structured can access the bucket contents.
  </Step>
</Steps>

### GitHub

To integrate with GitHub repositories:

<Steps>
  <Step>In Structured's 'Links' area, select 'GitHub'.</Step>
  <Step>
    You will be redirected to GitHub to authenticate and authorize the
    Structured application.
  </Step>
  <Step>
    After authentication, select the repositories you wish to integrate with
    Structured.
  </Step>
</Steps>
