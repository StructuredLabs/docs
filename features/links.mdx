## Current Links

Structured seamlessly integrates with a variety of services to enhance your experience. Hereâ€™s how you can connect each one:

### Local File System

To access files stored on your local file system:

1. Navigate to the 'Links' section in your Structured dashboard.
2. Select 'Local File System' and grant the necessary file access permissions prompted by your operating system.

### S3 Bucket

To connect to an Amazon S3 bucket:

1. Go to the 'Links' page on your Structured dashboard.
2. Choose 'S3 Bucket' and enter your AWS Access Key ID and Secret Access Key.
3. Provide the name of your S3 bucket and the corresponding AWS region.
4. Confirm the permissions to ensure Structured can access the bucket contents.

### GitHub

To integrate with GitHub repositories:

1. In Structured's 'Links' area, select 'GitHub'.
2. You will be redirected to GitHub to authenticate and authorize the Structured application.
3. After authentication, select the repositories you wish to integrate with Structured.

### Vercel

For Vercel deployment:

1. From the 'Links' menu in Structured, choose 'Vercel'.
2. Log in to your Vercel account to authenticate, if not already logged in.
3. Authorize Structured to interact with your Vercel projects for deployment capabilities.

### Modal

To use Modal's features:

1. In the 'Links' section, pick 'Modal'.
2. Authenticate with your Modal account to link it with Structured.
3. Once authenticated, you can access Modal's functionalities directly within Structured.
