---
title: "Deploy Block"
api: "POST https://api.structuredlabs.io/cli-deploy-service"
description: "This endpoint deploys a specific block, making it live and ready to use."
---

### Body

<ParamField query="apiToken" type="string" optional="false">
  The API token for authentication.
</ParamField>

<ParamField query="endpointInfo" type="object" optional="false">
  Information about the service endpoint.
</ParamField>

<ParamField query="flaskCode" type="string" optional="false">
  The Flask code associated with the service.
</ParamField>

<ParamField query="id" type="string" optional="false">
  The identifier for the service.
</ParamField>

<ParamField query="serviceName" type="string" optional="false">
  The name of the service.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the block deployment was successful. Returns `true` if successful, `false` if not.
</ResponseField>
<ResponseField name="block" type="object">
The details of the deployed block.

<Expandable title="Toggle object">
<ResponseField name="id" type="string">
  The unique identifier of the block.
</ResponseField>
<ResponseField name="endpoint" type="string">
  The URL endpoint where the block is now accessible.
</ResponseField>
<ResponseField name="status" type="string">
  The current status of the block, updated to 'deployed' after a successful operation.
</ResponseField>
</Expandable>
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api.structuredlabs.io/cli-deploy-service' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <api_key>'
```

</RequestExample>
