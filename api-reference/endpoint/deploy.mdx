---
title: "Deploy Block"
api: "POST https://api.structuredlabs.io/deploy-block"
description: "This endpoint deploys a specific block, making it live and ready to use."
---

### Body

<ParamField query="apiToken" type="string" optional="false">
  The API token for authentication.
</ParamField>

<ParamField query="blockData" type="object" optional="false">
  Block object to be deployed.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the block deployment was successful. Returns `true` if successful, `false` if not.
</ResponseField>
<ResponseField name="block" type="object">
The details of the deployed block.

<Expandable title="Toggle object">
<ResponseField name="id" type="string">
  The unique identifier of the block.
</ResponseField>
<ResponseField name="endpoint" type="string">
  The URL endpoint where the block is now accessible.
</ResponseField>
<ResponseField name="status" type="string">
  The current status of the block, updated to 'deployed' after a successful operation.
</ResponseField>
</Expandable>
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api.structuredlabs.io/deploy-block?apiToken={apiToken}&blockData={blockData}' \
--header 'Content-Type: application/json' \
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,
  "block": {
    "id": "blk_123456",
    "endpoint": "https://api.structuredlabs.io/blocks/blk_123456/deploy",
    "status": "deployed"
  }
}
```

</ResponseExample>
