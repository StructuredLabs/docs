---
title: "Create Block"
api: "POST https://api.structuredlabs.io/cli-create-service"
description: "This endpoint creates a new block from a natural language description."
---

### Body

<ParamField query="apiToken" type="string" optional="false">
  The API token for authentication.
</ParamField>

<ParamField query="prompt" type="string" optional="false">
  The natural language description of the API functionality you want to create.
</ParamField>

<ParamField query="serviceName" type="string" optional="false">
  The name of the service.
</ParamField>

<ParamField query="serviceId" type="string" optional="false">
  The identifier for the service.
</ParamField>

<ParamField query="isAuthorized" type="boolean" optional="true">
  Indicates if the user is authorized.
</ParamField>

<ParamField query="rateLimit" type="number" optional="true">
  The number of requests allowed per time frame for this API block.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the block creation was successful. Returns `true` if successful, `false` if not.
</ResponseField>
<ResponseField name="block" type="object">
The details of the newly created block.

<Expandable title="Toggle object">
<ResponseField name="id" type="string">
  The unique identifier of the block.
</ResponseField>
<ResponseField name="endpoint" type="string">
  The URL endpoint at which the new block can be accessed.
</ResponseField>
<ResponseField name="apiKey" type="string">
  The API key required for authorized access to the API block.
</ResponseField>
<ResponseField name="rateLimit" type="number">
  The rate limit configuration for the block, reflecting the number of allowed requests per time frame.
</ResponseField>
<ResponseField name="status" type="string">
  The current status of the block, such as 'pending', 'deployed', or 'failed'.
</ResponseField>
</Expandable>
</ResponseField>

<RequestExample>

```bash Example Request
curl --location --request POST 'https://api.structuredlabs.io/cli-create-service' \
--header 'Content-Type: application/json' \
--data-raw '{
    "prompt": "Calculate compound interest",
    "preConditions": {
        "principalRange": "[1000, 5000]",
        "rateRange": "[1, 10]",
        "timeRange": "[1, 5]"
    },
    "postConditions": {
        "resultFormat": "currency"
    },
    "rateLimit": 100
}'
```

</RequestExample>
