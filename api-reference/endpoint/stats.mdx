---
title: "Block Stats"
api: "GET https://api.structuredlabs.io/v1/blocks/{blockId}/stats"
description: "This endpoint retrieves usage and performance statistics for a specific block."
---

### Path Parameters

<ParamField path="blockId" type="string">
  The unique identifier of the block for which you want to retrieve statistics.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful. Returns `true` if successful,
  `false` if not.
</ResponseField>

<ResponseField name="stats" type="object">
  The usage and performance statistics of the specified block.
</ResponseField>

<RequestExample>

```bash Example Request
# Example Request to retrieve statistics for a specific block
curl --location --request GET 'https://api.structuredlabs.io/v1/blocks/{blockId}/stats' \
--header 'Authorization: Bearer <api_key>'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true,
  "stats": {
    "id": "blk_123456",
    "totalCalls": 1500,
    "successfulCalls": 1450,
    "failedCalls": 50,
    "averageResponseTime": 120
  }
}
```

</ResponseExample>
