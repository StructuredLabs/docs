---
title: "Get All Blocks"
api: "GET https://api.structuredlabs.io/get-user-blocks"
description: "This endpoint retrieves a list of all blocks created by the user."
---

### Body

<ParamField query="apiToken" type="string" optional="false">
  The API key for authentication.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful. Returns `true` if successful,
  `false` if not.
</ResponseField>

<ResponseField name="blocks" type="array">
  An array of objects, each representing an block and its details.
</ResponseField>

<RequestExample>

```bash Example Request
# Example Request to list all blocks using API token as a query parameter
curl --location --request GET 'https://api.structuredlabs.io/get-user-blocks?apiToken={apiToken}' \
--header 'Content-Type: application/json'
```

</RequestExample>
