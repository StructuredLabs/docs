---
title: "Get All Blocks"
api: "GET https://api.structuredlabs.io/v1/blocks"
description: "This endpoint retrieves a list of all blocks created by the user."
---

### Body

<ParamField query="page" type="number" optional="true">
  The page number for paginated results.
</ParamField>

<ParamField query="limit" type="number" optional="true">
  The number of items to display per page.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful. Returns `true` if successful,
  `false` if not.
</ResponseField>

<ResponseField name="blocks" type="array">
  An array of objects, each representing an block and its details.
</ResponseField>

<RequestExample>

```bash Example Request
# Example Request to list all blocks
curl --location --request GET 'https://api.structuredlabs.io/v1/blocks' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <api_key>'
```
