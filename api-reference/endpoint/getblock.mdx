---
title: "Get Block Details"
api: "GET https://api.structuredlabs.io/get-block/{blockName}"
description: "This endpoint retrieves the details of a specific block."
---

### Body

<ParamField query="apiToken" type="string" optional="false">
  The API key for authentication.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful. Returns `true` if successful,
  `false` if not.
</ResponseField>

<ResponseField name="block" type="object">

The detailed information of the block.

<Expandable title="Toggle object">

<ResponseField name="id" type="string">
  The unique identifier of the block.
</ResponseField>

<ResponseField name="name" type="string">
  The name given to the block.
</ResponseField>

<ResponseField name="endpoint" type="string">
  The URL endpoint where the block is accessible.
</ResponseField>

<ResponseField name="status" type="string">
  The current status of the block, such as 'active', 'inactive', or 'deploying'.
</ResponseField>

<ResponseField name="rateLimit" type="number" optional="true">
  The rate limit that has been set for the block, if any.
</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash Example Request
# Example Request to get details of a specific block
curl --location --request GET 'https://api.structuredlabs.io/get-block/{blockName}' \
--header 'Authorization: Bearer <api_key>'
--data-raw '{
    "blockId": "xyz",
}'
```

</RequestExample>

  }
