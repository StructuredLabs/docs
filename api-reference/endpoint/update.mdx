---
title: "Update Block"
api: "PUT https://api.structuredlabs.io/v1/blocks/{blockId}"
description: "This endpoint updates an existing block with new configurations or modifications."
---

### Path Parameters

<ParamField path="blockId" type="string">
  The unique identifier of the block to update.
</ParamField>

### Body

<ParamField body="prompt" type="string">
  An updated natural language description of the API functionality, if changes
  are needed.
</ParamField>

<ParamField body="preConditions" type="object" optional="true">
  Updated preconditions for the API functionality.
</ParamField>

<ParamField body="postConditions" type="object" optional="true">
  Updated post-conditions for the API functionality.
</ParamField>

<ParamField body="rateLimit" type="number" optional="true">
  The new rate limit configuration for the block, if changes are needed.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful. Returns `true` if successful,
  `false` if not.
</ResponseField>

<ResponseField name="block" type="object">

The updated contents of the block.

<Expandable title="Toggle object">

<ResponseField name="id" type="string">
  The unique identifier of the block.
</ResponseField>

<ResponseField name="prompt" type="string">
  The updated natural language prompt defining the API's functionality.
</ResponseField>

<ResponseField name="preConditions" type="object">
  The updated preconditions set for the block.
</ResponseField>

<ResponseField name="postConditions" type="object">
  The updated post-conditions set for the block.
</ResponseField>

<ResponseField name="rateLimit" type="number">
  The updated rate limit for the block.
</ResponseField>

<ResponseField name="status" type="string">
  The current status of the block after the update.
</ResponseField>

</Expandable>

</ResponseField>

<RequestExample>

```bash Example Request
# Example Request to update an block
curl --location --request PUT 'https://api.structuredlabs.io/v1/blocks/{blockId}' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <api_key>' \
--data-raw '{
    "prompt": "Updated prompt for calculating compound interest",
    "preConditions": {
        "principalRange": "[500, 5000]",
        "rateRange": "[1, 15]",
        "timeRange": "[1, 10]"
    },
    "postConditions": {
        "resultFormat": "currency"
    },
    "rateLimit": 50
}'
```

</RequestExample>
