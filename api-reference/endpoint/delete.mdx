---
title: "Delete API Block"
api: "DELETE https://api.structuredlabs.io/v1/blocks/{blockId}"
description: "This endpoint deletes an existing API block."
---

### Body

<ParamField body="blockId" type="string">
  The unique identifier of the API block you want to delete.
</ParamField>

<ParamField body="apiKey" type="string">
  The API key associated with your account, required to authenticate the request
  to delete the API block.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful. Returns `true` if successful,
  `false` if not.
</ResponseField>

<RequestExample>

```bash Example Request
# Example Request to delete an API block
curl --location --request DELETE 'https://api.structuredlabs.io/v1/blocks/{blockId}' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {apiKey}' \
--data-raw '{
    "blockId": "blk_123456",
    "apiKey": "api_key_789abc"
}'
```

</RequestExample>

<ResponseExample>

```json Response
{
  "success": true
}
```

</ResponseExample>
